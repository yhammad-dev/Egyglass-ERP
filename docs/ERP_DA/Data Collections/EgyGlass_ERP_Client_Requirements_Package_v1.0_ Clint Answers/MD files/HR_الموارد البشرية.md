# HR_الموارد البشرية


## الرؤية والمتطلبات

| الرؤية والمتطلبات - الموارد البشرية |  |  |  |  |  |
|---|---|---|---|---|---|
| EgyGlass - Human Resources | جهة الاتصال: راندا |  |  |  |  |  |
| المعرف | المتطلب | شرح المقصود | مثال توضيحي | مخرجات النظام | الأولوية |
| HR-R01 | سجل الموظفين | قاعدة بيانات مركزية لجميع موظفي الشركة تحتوي على البيانات الأساسية والوثائق | مثال: كل موظف له ملف إلكتروني يضم: العقد، المؤهل، الراتب، الإجازات، الشهادات | سجل موظفين إلكتروني شامل | ضروري |
| HR-R02 | إدارة الإجازات | نظام لتقديم واعتماد وإدارة إجازات الموظفين إلكترونيًا | مثال: موظف يقدم إجازة مرضية عبر النظام، مديره المباشر يعتمد، وراندا تشوف الرصيد المتبقي | نظام إجازات إلكتروني مع رصد الأرصدة | ضروري |
| HR-R03 | ربط الموظفين بالمشاريع | تحديد الموظفين العاملين على كل مشروع وتتبع ساعات العمل | مثال: إسلام السيد يدخل فريقه على مشروع برج كليوباترا - كل فني مسجل على المشروع | شاشة توزيع الموظفين على المشاريع | ضروري |
| HR-R04 | إدارة العقود | أرشفة وإدارة عقود الموظفين إلكترونيًا مع تتبع تواريخ التجديد | مثال: النظام يرسل إشعار قبل 30 يومًا من انتهاء عقد موظف | أرشيف عقود إلكتروني مع تنبيهات | ضروري |
| HR-R05 | تقييم الأداء | نظام لتقييم أداء الموظفين بشكل دوري وربطه بالمشاريع | مثال: بعد كل مشروع، مدير المشروع يقيم أداء كل فرد في الفريق (1-5) | نظام تقييم أداء مع تقارير | مهم |
| HR-R06 | التقارير والإحصائيات | تقارير عن عدد الموظفين، الإجازات، التوزيع على الأقسام، التكاليف | مثال: تقرير شهري يظهر عدد الموظفين في كل قسم ومعدل الإجازات | لوحة مؤشرات موارد بشرية | متوسط |

## أسئلة BRD - الموارد البشرية

| BRD - كل المعلومات المطلوبة من الموارد البشرية |  |  |  |  |  |  |
|---|---|---|---|---|---|---|
| # | السؤال | شرح المطلوب مع مثال | الأولوية | حالة الرد | رد المسؤول | ملاحظات |
| 2 | هل يوجد حالياً نظام لإدارة شؤون الموظفين (حضور، إجازات، رواتب)؟ | مثال: لا يوجد — كل شيء Excel. هل مطلوب إدراج HR كامل داخل النظام ولا يبقى خارجي؟ | 🔴 HIGH | ⏳ | نعم |  |
| 3 | ما هي أنواع الإجازات المطلوبة وكيف تُحتسب؟ | مثال: إجازة سنوية 21 يوم، مرضية 7 أيام مدفوعة، عارضة 3 أيام. هل توجد إجازات خاصة بالقطاع (مثل عيد العمال للمصانع)؟ | 🔴 HIGH | ⏳ | 21يوم اجازة سنوية ( 14 يوم اعتيادي - 7 ايام عارضة) |  |
| 4 | هل نظام الحضور والانصراف يدوي أم بصمة (Face ID / Fingerprint)؟ | مثال: نحتاج ربط النظام الجديد بأجهزة البصمة الحالية. أم نكتفي بإدخال يدوي من المدير؟ | 🟡 MEDIUM | ⏳ | الاثنين حاليا ( يدوي - بصمة) |  |
| 5 | هل يحتاج النظام حساب كشوف رواتب (Payroll)؟ | مثال: الراتب الأساسي + بدل نقل + بدل سفر + خصومات تأخير + ساعات إضافي + تأمينات اجتماعية + ضريبة مرتبات. | 🔴 HIGH | ⏳ | نعم |  |
| 6 | ما هي قواعد الحوافز والعمولات؟ | مثال: عمولة لكل مندوب مبيعات (نسبة من الربح/إجمالي المبيعات). للمهندسين/الفنيين حوافز إنتاجية؟ | 🟡 MEDIUM | ⏳ |  |  |
| 7 | هل يوجد هيكل تنظيمي (Organizational Chart) محدد مطلوب؟ | مثال: شجرة تنظيمية — كل قسم وتابعيته. هل يحتاج النظام يعرضها بشكل تفاعلي؟ | 🟡 MEDIUM | ⏳ | نعم |  |
| 8 | هل مطلوب ملف إلكتروني لكل موظف (Personnel File)؟ | مثال: العقد، المؤهل، الشهادات، التقييم السنوي، إجازات، قرارات جزاءات. | 🟡 MEDIUM | ⏳ | نعم |  |
| 9 | هل يحتاج النظام إدارة تدريب (Training) للموظفين؟ | مثال: تسجيل دورات تدريبية، متابعة شهادات، تذكير بتجديد الشهادات (مثل سلامة وصحة مهنية). | 🟢LOW | ⏳ | نعم |  |
| 10 | ما هي تقارير HR المطلوبة دورياً؟ | مثال: عدد الموظفين لكل قسم، نسبة الغياب، إجازات الشهر، تكاليف الرواتب الشهرية. | 🟡 MEDIUM | ⏳ | نعم |  |
