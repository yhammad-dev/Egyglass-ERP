# PRC_المشتريات والمخازن


## الرؤية والمتطلبات

| الرؤية والمتطلبات - المشتريات والمخازن |  |  |  |  |  |
|---|---|---|---|---|---|
| EgyGlass - Procurement & Warehouses | جهة الاتصال: شكري |  |  |  |  |  |
| المعرف | المتطلب | شرح المقصود | مثال توضيحي | مخرجات النظام | الأولوية |
| PRC-R01 | استلام أمر تصنيع كامل | نظام يتيح استلام أمر تصنيع متكامل: رسم معاينات + رسم مكتب فني + بطاقة إكسسوار + إضافات | مثال: شكري يستلم أوردر رقم ORD-2026-001 مع 3 مرفقات وبيان بالمكونات | شاشة أوامر تصنيع مع المرفقات كاملة | ضروري |
| PRC-R02 | مرحلة مراجعة قبل التصنيع | خطوة مراجعة إجبارية للتأكد من مطابقة الرسمين مع بطاقة الإكسسوار | مثال: قبل إرسال الأوردر للمصنع، النظام يطلب مراجعة من شكري أو مسؤول المراجعة | شاشة مراجعة مع إمكانية قبول/رفض | ضروري |
| PRC-R03 | رفض الأوردر للمراجعة | إمكانية إعادة الأوردر للمكتب الفني في حال عدم التطابق | مثال:قسم المعينات  لاحظ اختلاف في المقاسات بين الرسم وبطاقة الإكسسوار ويعيد الأوردر للمكتب الفني | زر إعادة مع سبب الرفض | ضروري |
| PRC-R04 | إرسال الأوردر للمصنع | إرسال أمر التصنيع للمصنع المختار بعد إتمام المراجعة | مثال: بعد المراجعة، شكري يختار المصنع من قائمة ويضغط 'إرسال للتصنيع' | نظام إرسال مع تأكيد الاستلام | ضروري |
| PRC-R05 | تحديد تاريخ التسليم المتوقع | إدخال تاريخ متوقع للاستلام من المصنع وتتبع الالتزام به | مثال: المصنع يؤكد تسليم 15 يونيو، النظام يحسب تلقائيًا الأيام المتبقية | حقل تاريخ متوقع مع عداء (Countdown) | ضروري |
| PRC-R06 | إشعارات التأخير | تنبيهات عند تأخير المصنع عن التاريخ المتوقع | مثال: لو المصنع تأخر يومين عن 15 يونيو، النظام يرسل إشعار لشكري وإسلام | نظام إشعارات تلقائية للتأخير | ضروري |
| PRC-R07 | إدارة بطاقة الإكسسوار | تجهيز وإرسال بطاقة الإكسسوار للاعتماد ثم للمصنع | مثال: شكري يجهز بطاقة الإكسسوار، يرسلها للاداره  للاعتماد، ثم للمصنع | نظام بطاقة إكسسوار إلكتروني | ضروري |
| PRC-R08 | إشعار الجاهزية للتركيبات | إرسال إشعار تلقائي لقسم التركيبات عند اكتمال التصنيع | مثال: لما الأوردر يوصل للمخزن، النظام يبعت إشعار لإسلام السيد 'الأوردر جاهز' | إشعار جاهزية مع تفاصيل الأوردر | ضروري |
| PRC-R09 | إشعار المناديب | إرسال إشعارات للمندوبين بمواعيد استلام المواد من المصانع | مثال: المندوب يستلم إشعار على الموبايل 'استلام من مصنع الزجاج غدًا 10 صباحًا' | إشعارات موبايل للمندوبين | مهم |
| PRC-R10 | تسجيل التكاليف الكاملة | تسجيل جميع التكاليف: طلب شراء، طلب تصنيع، مستهلكات، نقل خارجي | مثال: شكري يدخل تكلفة الشحن 2,000 جنيه ويربطها بالمشروع | نظام تكاليف لكل مشروع | ضروري |
| PRC-R11 | لوحة تكاليف المشروع | عرض التكلفة الإجمالية لكل مشروع بشكل ظاهر | مثال: شاشة تعرض مشروع برج كليوباترا بتكلفة إجمالية 850,000 جنيه مفصلة | لوحة تكاليف مع تفصيل | ضروري |

## أسئلة BRD - المشتريات

| BRD - كل المعلومات المطلوبة من المشتريات والمخازن (شكري) |  |  |  |  |  |  |
|---|---|---|---|---|---|---|
| # | السؤال | شرح المطلوب مع مثال | الأولوية | حالة الرد | رد المسؤول | ملاحظات |
| 2 | إشعارات التأخير: جوه النظام ولا إيميل ولا واتساب؟ | مثال: الأوردر اتأخر → النظام يبعت Notification + إيميل للمدير + واتساب للعميل | 🟡 MEDIUM | ⏳ | داخل النظام  |  |
| 3 | هل في مصانع محددة ثابتة ولا المصنع يختلف حسب الأمر؟ | مثال: 3 مصانع ثابتين. النظام يحتفظ ببياناتهم وتقييمهم | 🔴 HIGH | ⏳ | في مصانع ثابته وفي مصانع بتختلف حسب الامر  |  |
| 4 | ربط أوتوماتيكي بين المخزون والأوردرات الجديدة؟ | مثال: المكتب الفني طلب خامة X. النظام يشوف المخزون: موجود 50، المحتاج 30 — يشتري ولا يجهز؟ | 🔴 HIGH | ⏳ | حاليا مافيش مخزون بسب ممكن بعدين يكون في  |  |
| 5 | تقارير التكاليف: متاحة لمين غير شكري؟ | مثال: المكتب الفني للتسعير، كريم لمتابعة المشروع، المدير لإجمالي التكاليف | 🟡 MEDIUM | ⏳ | شكري فقط  |  |
| 6 | هل في كتالوج موحد للمنتجات والخامات والأسعار؟ | مثال: الزجاج (6مم-8مم-10مم)، الإطارات (ألوميتال-خشب)، الإكسسوارات. لكل منتج سعر أساسي + وحدة قياس | 🟡 MEDIUM | ⏳ | هو مش كتالوج  هي قايمه اسعار متغير كل مده  |  |
| 7 | وحدات قياس ثابتة للمخازن؟ باركود للتتبع؟ | مثال: متر - قطعة - لفة. الباركود: لا حالياً لكن مطلوب مستقبلاً | 🟢 LOW | ⏳ | لا مافيش للمخازن باركود |  |
| 8 | تقييم المصانع بناءً على الأخطاء والتأخير؟ | مثال: مصنع X عنده 3 أخطاء في آخر 10 أوردرات. النظام يحسب النسبة | 🟢 LOW | ⏳ | تمام  |  |
| 9 | ما هي التقارير الدورية المطلوبة للمشتريات؟ | مثال: تقرير مخزون أسبوعي، تأخير توريد شهري، تكاليف مشتريات شهري | 🟡 MEDIUM | ⏳ | تكاليف مشتريات لكل عمليه لوحدها +اجمالي شهري للكل  |  |
| 10 | مصاريف النقل: بتتضاف إزاي؟ نقدي - حساب مورد - شهري؟ | مثال: كل أمر تصنيع ليه تكلفة نقل. هل بتتضاف يدوي لكل أمر؟ ولا في حساب شهرى معين للنقل؟ | 🟢 LOW | ⏳ | يدوي لكل مليه  |  |
