# تعليمات استيراد المخرجات

---

## 1. مستند BRD — Word / Google Docs

**الملف:** `BRD\BRD_EgyGlass_Internal_Management_System_v1.0.md`

**الطريقة:**
1. افتح Google Docs (أو Word)
2. انسخ محتوى الملف بالكامل
3. استخدم: **Edit > Paste** (يحافظ على التنسيق)
4. طبّق:
   - `# عناوين` ← **Heading 1**
   - `## عناوين` ← **Heading 2**
   - `### عناوين` ← **Heading 3**
   - الجداول: Select All ثم **Insert > Table**
5. أضف شعار EgyGlass في الصفحة الأولى
6. املأ خانة **المعتمِد** في صفحة العنوان وسجل المراجعة

### المخطط 3: Use Case Diagram
**الملف:** `Diagrams\03_Use_Case_Diagram.xml`

**طريقة الاستيراد لكل مخطط:**
1. افتح https://app.diagrams.net
2. اختر **Extras > Edit Diagram** (على اليسار)
3. امسح المحتوى الموجود بالكامل
4. الصق محتوى ملف XML كاملًا
5. اضغط **OK**
6. المخطط سيظهر كاملًا — يمكنك تعديله أو تصديره كصورة

**نصيحة:** إذا كانت الأحرف العربية لا تظهر، غيّر الخط إلى Arial أو Noto Sans Arabic

## 3. جدول User Stories — Excel / Google Sheets

**الملف:** `User Stories\User_Stories_Table.tsv`

**الطريقة:**
1. افتح Excel (أو Google Sheets)
2. اختر الخلية **A1**
3. افتح ملف `.tsv` في Notepad، انسخ المحتوى
4. استخدم **Paste** مباشرة
5. يفصل تلقائيًا:
   - العمود A: ID
   - العمود B: العنوان
   - العمود C: User Story
   - العمود D: الأولوية
   - العمود E: معيار القبول
   - العمود F: الحالة

**تحويل لجدول رسمي:**
- حدد كل الخلايا (Ctrl+A)
- Insert > Table
- طبّق فلترة حسب الأولوية أو الحالة

**ملف Excel بديل (بفاصلة):**
إذا احتجت ملف `.csv` بدل `.tsv`، افتح الملف في Notepad واستبدل كل `\t` بفواصل `,` واحفظ بامتداد `.csv`
